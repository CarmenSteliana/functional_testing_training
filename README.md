# Functional Testing Training & Quiz

This project is an interactive e-learning web page designed for new testers joining the peviitor.ro project. Created with the help of Gemini Canvas, it introduces the core concepts of functional testing in a structured and easy-to-understand format. The training includes practical examples, clear explanations, and an interactive quiz section for knowledge verification, helping new team members quickly understand testing fundamentals and project-specific quality assurance practices. 

## What I Built

I developed a single-page web application that explains the fundamentals of functional testing and includes practical examples, testing techniques, and an interactive quiz. The project was created using Gemini Canvas together with HTML, CSS, and JavaScript.

## Project Chapters

The application is divided into several educational sections:

### 1. What is Functional Testing
An introduction to functional testing, including its definition and role in software quality assurance.

### 2. Purpose of Functional Testing
Explanation of why functional testing is important and how it helps verify that an application works according to requirements.

### 3. Black Box Testing Concept
Presentation of the black box testing approach, where the tester validates the system functionality without accessing the internal code structure.

### 4. Black Box Techniques + Examples
Description of common black box testing techniques, such as:
- Equivalence Partitioning
- Boundary Value Analysis
- Decision Table Testing
- State Transition Testing

This section also includes practical examples for each technique.

### 5. Functional Testing Process
Overview of the main stages involved in functional testing:
- Requirement analysis
- Test planning
- Test case creation
- Test execution
- Bug reporting
- Retesting and validation

### 6. Main Types and Levels of Functional Testing
Presentation of the primary testing categories and levels, including:

#### Testing Levels
- Unit Testing
- Integration Testing
- System Testing
- Acceptance Testing

#### Main Types of Functional Testing
- Smoke Testing
- Sanity Testing
- Regression Testing
- Positive Testing
- Negative Testing
- Re-testing

This section explains the purpose of each testing level and testing type, together with examples of how they are used in real software development and quality assurance processes.

### 7. Practical Examples of Functional Testing
Real-life examples that demonstrate how functional testing is applied in software applications.

### 8. Tools Used
Presentation of tools commonly used in functional testing and automation.

### 9. Quiz Section
An interactive quiz designed to verify the user’s understanding of the presented concepts.

## Features

- Educational content organized into chapters
- Interactive and user-friendly design
- Quiz with multiple-choice questions
- Practical examples for better understanding
- Responsive single-page layout

## Technologies Used

- HTML
- Tailwind CSS
- JavaScript
- React
- JSX
- Babel
- Gemini Canvas

  The project is built as a single HTML file, but the main application logic is written in JavaScript using React and JSX. Tailwind CSS is used for styling, while Babel is included to compile JSX directly in the browser.

## Role of Gemini Canvas

Gemini Canvas was used to help structure the application, generate content ideas, organize the interface, and assist with the development of the interactive quiz section.

## Project Purpose

The purpose of this project is to provide a clear and interactive introduction to functional testing concepts while helping users reinforce their knowledge through examples and quizzes.

## How to Use

1. Open the HTML file in a web browser.
2. Navigate through each chapter to learn about functional testing.
3. Review the examples and testing techniques.
4. Complete the quiz to test your understanding.

## Conclusion

This project demonstrates how educational web applications can be created quickly and effectively using Gemini Canvas together with HTML, CSS, and JavaScript.
